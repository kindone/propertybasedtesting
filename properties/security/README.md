# Security Properties

Strategies for testing security properties.

## Properties

1. **[Authentication](01_authentication.md)** - Only authorized users can access
2. **[Authorization](02_authorization.md)** - Users can only access permitted resources
3. **[Confidentiality](03_confidentiality.md)** - Sensitive data not leaked
4. **[Integrity](04_integrity.md)** - Data not tampered with

## Overview

Security properties ensure system security:
- **Authentication**: Verify identity, prevent unauthorized access
- **Authorization**: Enforce access control, prevent privilege escalation
- **Confidentiality**: Protect sensitive data, prevent leaks
- **Integrity**: Detect tampering, maintain data integrity
